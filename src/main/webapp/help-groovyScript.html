Execute a groovy script. The script should return boolean value. The job will not run if script evaluation returned false. The script is always run on the master, but solution to run closure on slave node is provided.

Pre-imported packages:
jenkins.model.*, hudson.model.*,
hudson.util.*, hudson.remoting.*,
groovyx.net.http.*, net.sf.json.*

Predefined variables:
build The current MultiJob build.
listener A BuildListener.
out A PrintStream (listener.getLogger()).
Also, environment variables of build are injected.

Predefined methods:
net.sf.json.JSON getJson(String url) //get JSON for specified URL. If case of non OK response code or parse exception empty JSON will be returned.
runScriptOnNode(String node, String script //run groovy script on the specified node (by node name). Packages and variables aren't imported but can be constants can be injected to the script source.
runScriptOnCurrentNode(String script) //run script on the current node.
